üíª Code & Methodology

The project followed a standard machine learning workflow, with a specific focus on handling data imbalance to ensure the model could effectively identify rare fraudulent cases.

1. Data Loading & Preparation

The dataset was loaded into a Pandas DataFrame, and features (`X`) and the target variable (`y`) were separated. The `Class` variable was identified as the target. 

import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
from imblearn.over_sampling import SMOTE
import joblib
     
data = pd.read_csv("creditcard.csv")
data = data.dropna(subset=['Class'])
X = data.drop(columns=['Class'])
y = data['Class']

2. Handling Data Imbalance (SMOTE)

To address the imbalanced data, I used SMOTE to generate synthetic examples of the minority class (fraudulent transactions). This balanced the dataset and prevented the model from being biased towards the majority class. 

smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X, y)

3. Model Training & Persistence

I chose a Random Forest Classifier for its ability to handle complex datasets and its high accuracy. After training the model, I saved it to a `.pkl` file using `joblib` for easy reuse without retraining. 

X_train, X_test, y_train, y_test = train_test_split(X_resampled, y_resampled, test_size=0.2, random_state=42)
     
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
     
joblib.dump(model, "fraud_model.pkl")

4. Model Evaluation

The model's performance was evaluated on the test set using key metrics, particularly Recall, which is crucial for fraud detection as it measures the model's ability to identify all positive cases (fraudulent transactions). 

y_pred = model.predict(X_test)
     
accuracy = accuracy_score(y_test, y_pred)
precision = precision_score(y_test, y_pred)
recall = recall_score(y_test, y_pred)
f1 = f1_score(y_test, y_pred)
     
print(f"Accuracy: {accuracy:.4f}")
print(f"Precision: {precision:.4f}")
print(f"Recall: {recall:.4f}")
print(f"F1-score: {f1:.4f}")
