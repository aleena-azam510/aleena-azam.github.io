Seamless video conferencing, redefined.

üìù About This Project

VisionMeet is a modern video conferencing application built to provide a smooth and reliable platform for virtual meetings, collaborations, and personal calls. It leverages modern web technologies to enable real-time communication directly in the browser.

‚ú® Core Features

  • Real-time Video & Audio: High-quality, low-latency communication.
  • Dynamic Meeting Rooms: Users can create or join meetings with a unique meeting ID.
  • Screen Sharing: Share your entire screen or a specific application window.
  • Chat Functionality: Text chat within each meeting room.
  • Responsive UI: Seamless experience across desktop and mobile devices.

‚öôÔ∏è Technical Approach

  • WebRTC: Utilized the Web Real-Time Communication API for peer-to-peer video and audio streaming without requiring a server to relay the media.
  • Socket.IO: Implemented a server using Socket.IO to handle signaling, meeting room management, and chat messaging.
  • Express.js: A minimal web framework to serve the front-end application and manage server-side logic.

üß≠ How to Use

Follow these simple steps to start a new meeting or join an existing one:

  1. Create a Meeting: Click "Create a Meeting". A unique meeting ID will be generated.
  2. Join a Meeting: Click "Join a Meeting" and enter the meeting ID provided by the host.
  3. Start Communicating: Once connected, you can see and hear other participants, share your screen, and use the chat.

üíª Code Snippet (Example)

Here's a brief example of the client-side code that initiates a WebRTC connection and handles video streams. 

const socket = io('/');
const videoGrid = document.getElementById('video-grid');
const myPeer = new Peer(undefined, {
    host: '/',
    port: '3001'
});

myPeer.on('open', id => {
    socket.emit('join-room', ROOM_ID, id);
});

navigator.mediaDevices.getUserMedia({
    video: true,
    audio: true
}).then(stream => {
    addVideoStream(myVideo, stream);
    myPeer.on('call', call => {
        call.answer(stream);
        const video = document.createElement('video');
        call.on('stream', userVideoStream => {
            addVideoStream(video, userVideoStream);
        });
    });
    socket.on('user-connected', userId => {
        connectToNewUser(userId, stream);
    });
});

function addVideoStream(video, stream) {
    video.srcObject = stream;
    video.addEventListener('loadedmetadata', () => {
        video.play();
    });
    videoGrid.append(video);
}
