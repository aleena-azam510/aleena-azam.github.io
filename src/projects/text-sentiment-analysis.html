Text Sentiment Analysis

An end-to-end case study of a machine learning-powered web application.

üìù Project Overview

This project is a web application that analyzes a given text to determine its sentiment (positive, negative, or neutral). It showcases the full pipeline of a machine learning project, from data preprocessing to model training, evaluation, and deployment.

Technologies: Python, Scikit-learn, Flask, NLTK, HTML, CSS

üíª Code & Methodology

The core of the project is a Logistic Regression model trained on a large dataset of movie reviews. Below is the step-by-step process of how the model was built and evaluated.

1. Importing Libraries & Data

The first step was to import all the necessary libraries for data manipulation, text preprocessing, and model training. The dataset was read from a CSV file. 

import pandas as pd
import re
import string
import nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize
from nltk.stem import WordNetLemmatizer
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

df = pd.read_csv('IMDB Dataset.csv', on_bad_lines='skip', engine="python")

2. Text Preprocessing

A critical step in any NLP project is cleaning the text data. I created a function to convert text to lowercase, remove HTML tags and punctuation, and then tokenize, remove stopwords, and lemmatize the words. 

def preprocess_text(text):
    # Convert to lowercase
    text = text.lower()
    # Remove HTML tags
    text = re.sub(r'<.*?>', '', text)
    # Remove punctuation
    text = text.translate(str.maketrans('', '', string.punctuation))
    # Tokenize words
    words = word_tokenize(text)
    # Remove stopwords
    words = [word for word in words if word not in stopwords.words('english')]
    # Lemmatization
    lemmatizer = WordNetLemmatizer()
    words = [lemmatizer.lemmatize(word) for word in words]
    return ' '.join(words)

df['cleaned_review'] = df['review'].apply(preprocess_text)

3. Data Preparation & Vectorization

The sentiment labels were converted to a binary format (1 for positive, 0 for negative). The data was split into training and testing sets, and then the text was converted into numerical features using TF-IDF (Term Frequency-Inverse Document Frequency), which is a powerful technique for representing the importance of words in a document. 

df['sentiment'] = df['sentiment'].map({'positive': 1, 'negative': 0})
X_train, X_test, y_train, y_test = train_test_split(df['cleaned_review'], df['sentiment'], test_size=0.2, random_state=42)

vectorizer = TfidfVectorizer(max_features=5000)
X_train_tfidf = vectorizer.fit_transform(X_train)
X_test_tfidf = vectorizer.transform(X_test)

4. Model Training & Evaluation

I chose Logistic Regression for its efficiency and strong performance on text classification tasks. The model was trained and its performance was evaluated using standard metrics. 

model = LogisticRegression()
model.fit(X_train_tfidf, y_train)
y_pred = model.predict(X_test_tfidf)

accuracy = accuracy_score(y_test, y_pred)
precision = precision_score(y_test, y_pred)
recall = recall_score(y_test, y_pred)
f1 = f1_score(y_test, y_pred)

print(f"Accuracy: {accuracy:.4f}")
print(f"Precision: {precision:.4f}")
print(f"Recall: {recall:.4f}")
print(f"F1-score: {f1:.4f}")
